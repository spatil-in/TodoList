## 📝 Todo List App

A simple Todo List application built with React and Context API. It helps manage daily tasks with features like adding, editing, deleting, and marking tasks as completed.

---

## 🚀 Features

* Add new tasks
* Edit existing tasks
* Delete tasks
* Mark tasks as completed
* Global state management using Context API
* Clean and responsive UI

---

## 🛠️ Tech Stack

* React
* JavaScript
* Context API
* Tailwind CSS

---

## ⚙️ How to Run

```bash id="s2j991"
git clone https://github.com/your-username/todo-context-app.git
cd todo-context-app
npm install
npm run dev
```
